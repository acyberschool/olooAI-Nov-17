## Slide 5: Under the Hood (Tech Stack)

We utilize a thick-client, serverless architecture powered by the latest **Google Gemini 2.5** models.

1.  **The Router Brain (Gemini 2.5 Flash):** A proprietary intent-classification engine that maps natural language to database schema. It knows the difference between a "Task," a "CRM Note," and a "Strategic Question."
2.  **Real-Time Voice (Gemini Live API):** Low-latency WebSocket connection allowing interruptible, human-like conversation.
3.  **Generative Engine:**
    *   **Video:** Veo 3.1 for marketing assets.
    *   **Images:** Nano Banana for social content.
4.  **Multi-Tenant Backend:** Supabase (PostgreSQL) with Row Level Security (RLS) for enterprise-grade data isolation.
